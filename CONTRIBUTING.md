# 🤝 Contributing to KiruhiPlace

We love your help! Follow these 3 simple steps to contribute:

### 1. Find or Create an Issue
Check the [Issues](link-buraya) tab to see what needs fixing. If you have a new idea, open a new issue first.


### 2. Fork & Branch
Fork the repo and create a branch for your feature:
`git checkout -b feature/AmazingFeature`

### 3. Pull Request (PR)
Submit your PR with a clear description. Once reviewed and tested, it will be merged!
🚀 **Done!**
